## 🏪 商業賬號功能（核心新增功能）

### 1. 商家中心儀表板
**路由**: `/merchant/dashboard`
- 只有商戶和管理員可以訪問
- **信息卡片顯示**：
  - 總商品數
  - 已上架商品數
  - 已下架商品數

- **商品列表表格**：
  - 顯示所有用戶的商品
  - 顯示商品價格和狀態（已上架/已下架）
  - 允許批量管理商品

### 2. 編輯商品功能
**路由**: `/product/<product_id>/edit` [GET, POST]
- 只有商品所有者或管理員可以編輯
- **可修改參數**：
  - 商品名稱
  - 商品描述
  - 商品價格
- 提交後自動返回商家中心

### 3. 下架/上架產品
**路由**: 
- `/product/<product_id>/deactivate` [POST] - 下架產品
- `/product/<product_id>/activate` [POST] - 上架產品

- 只有商品所有者或管理員可以操作
- 下架後的產品不會在首頁顯示
- 商家可以隨時重新上架已下架的產品

### 4. 權限控制
- 確保只有商品所有者和管理員能進行操作
- 其他用戶嘗試訪問會收到權限警告

## 📝 數據模型更改

### Product 模型
```python
# 新增字段
is_active = db.Column(db.Boolean, default=True, nullable=False)
```
- **目的**：標記產品是否上架
- **默認值**：True（新商品默認上架）
- **影響**：首頁只顯示 `is_active=True` 的產品

## 🔄 工作流程示例

### 商戶工作流：
1. 以商戶身份登入
2. 點擊導航欄 "商家中心"
3. 查看所有已上架和已下架的商品
4. 點擊 "編輯" 修改商品信息
5. 點擊 "下架" 移除某個商品
6. 點擊 "添加新商品" 上架新商品

### 購買者工作流：
1. 瀏覽首頁商品列表（只顯示已上架商品）
2. 點擊商品查看詳情
3. 加入購物車
4. 聯絡賣家或進行評分
