# 🚀 微型商店 - 快速入門指南

## 新增功能概述

您的應用現在包含以下重要新功能：

### 1️⃣ **商業賬號 - 商家中心**
- 📍 **位置**: 導航欄 → 商家中心（僅商戶可見）
- 🎯 **功能**: 管理您上架的所有商品
- 📊 **信息卡片**: 顯示商品總數、已上架、已下架統計

### 2️⃣ **商品編輯功能**
- 📍 **位置**: 商家中心 → 編輯按鈕
- 🎯 **功能**: 修改商品名稱、描述、價格
- ✨ **特點**: 實時更新，即刻生效

### 3️⃣ **產品上架/下架管理**
- 📍 **位置**: 商家中心 → 下架/上架按鈕
- 🎯 **功能**: 控制商品是否在首頁展示
- ⚡ **效果**: 下架後不會出現在購買者的商品列表中

### 4️⃣ **優雅的 UI 風格**
- 🎨 **按鈕設計**: 所有按鈕採用低調優雅的配色
- 🏠 **主頁 Logo**: 導航欄品牌改進，添加視覺標記
- 🔤 **交互效果**: 平滑的懸停效果和按鈕動畫

## 🛠️ 操作步驟

### 對於商戶

#### 第一次使用
1. 點擊導航欄的 **"註冊"**
2. 填寫用戶名、郵箱、密碼
3. ✅ **勾選 "註冊為商戶帳號"**（重要！）
4. 點擊 **"Register"** 完成註冊

#### 添加商品
1. 登入後，點擊導航欄 **"新增商品"**
2. 填寫商品信息（名稱、描述、價格）
3. 點擊 **"Add Product"** 提交
4. 商品將立即上架

#### 管理商品
1. 點擊導航欄 **"商家中心"**
2. 查看您的所有商品
3. 可以進行以下操作：
   - **查看**: 點擊 "查看" 預覽商品詳情
   - **編輯**: 點擊 "編輯" 修改商品信息
   - **下架**: 點擊 "下架" 從首頁移除商品
   - **上架**: 點擊 "上架" 將已下架商品恢復

### 對於購買者

#### 瀏覽商品
1. 進入首頁
2. 瀏覽所有已上架商品
3. 點擊 **"查看詳情"** 了解更多

#### 購買流程
1. 點擊商品的 **"加入購物車"**
2. 進入購物車查看已選商品
3. 調整數量或移除商品
4. 點擊 **"結帳"** 完成購買（如適用）

#### 與賣家聯絡
1. 在商品詳情頁面點擊 **"聯絡賣家"**
2. 發送訊息給商家
3. 在導航欄 **"訊息"** 中查看所有對話

#### 評分與評論
1. 在商品詳情頁面向下滾動到 **"評論與評分"** 區域
2. 選擇星級評分（1-5 顆星）
3. 撰寫評論（可選）
4. 點擊 **"Submit Review"** 提交

## 💾 數據模型更新

### Product 表新增字段
```python
is_active: Boolean = True  # 商品是否上架
```

### 影響
- 新商品默認為已上架 (`is_active=True`)
- 首頁查詢自動過濾只顯示 `is_active=True` 的商品
- 已下架商品仍然存儲在數據庫中，可隨時恢復

## ❓ 常見問題

**Q: 我已經註冊了普通用戶，怎樣升級為商戶？**
A: 目前需要重新註冊為商戶。未來版本可能會添加升級功能。

**Q: 下架的商品會被刪除嗎？**
A: 不會！下架的商品仍然保存在數據庫中，您可以隨時重新上架。

**Q: 商品編輯後購買者能立即看到嗎？**
A: 是的，修改立即生效，無需重新加載。

**Q: 我能編輯其他賣家的商品嗎？**
A: 不能。系統會阻止非所有者的編輯操作。
