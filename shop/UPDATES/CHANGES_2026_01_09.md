## 🚀 如何使用新功能

### 註冊新帳戶
```
1. 點擊 "註冊" 按鍵
2. 填寫用戶名、郵箱、密碼
3. 選擇是否為商戶
4. 提交表單
```

### 查看個人頁面
```
1. 登錄後，點擊導航欄中的用戶名
2. 進入個人頁面查看統計信息
3. 點擊 "編輯個人資料" 修改信息
```

### 管理通知
```
1. 點擊導航欄的 "訊息" 按鍵（帶有紅色徽章）
2. 查看所有通知
3. 標記為已讀或刪除
```

### 添加商品（商戶）
```
1. 登錄商戶帳戶
2. 點擊 "商家中心"
3. 在商家中心添加、編輯或下架商品
```
