## 系統架構說明

### 資料模型
- User：使用者資料，包含類型區分（顧客/商戶）
- Product：商品資訊，關聯到商戶
- Review：商品評論，包含評分
- CartItem：購物車項目，關聯用戶與商品
- Message：用戶間訊息，記錄發送者與接收者

### 主要功能
1. 用戶系統
   - 註冊：可選擇顧客或商戶帳號
   - 密碼：支援顯示/隱藏切換
   - 權限：根據用戶類型限制功能

2. 商品管理
   - 上架：限商戶與管理員
   - 價格：使用港幣（HKD）
   - 評論：支援文字評論與星級評分

3. 購物功能
   - 購物車：支援加入、修改數量、移除
   - 金額：自動計算小計與總計
   - 介面：清晰的購物車管理頁面

4. 溝通系統
   - 介面：獨立的訊息中心
   - 功能：支援用戶間直接對話
   - 整合：從商品頁可直接聯絡賣家

### 使用技術
- 後端：Flask + SQLite + SQLAlchemy
- 前端：Bootstrap 5 + Bootstrap Icons
- 認證：Flask-Login
- 表單：Flask-WTF
- 樣式：自適應響應式設計
